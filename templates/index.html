About Our Blockchain Charity Payment System

The Blockchain Charity Payment System is a decentralized donation platform built using Django and Web3 technology. It integrates Ethereum-based transactions with a secure backend to ensure that every donation is verifiable, immutable, and transparent.

Each contribution made by a donor is validated through the blockchain network (Ganache test environment), recorded with a unique transaction hash, and securely stored in the system to prevent tampering or fraud.

Ensuring transparency, accountability, and trust in every donation.

  • Ethereum transaction verification using Web3 integration.
  • Automatic validation of trust wallet addresses.
  • Real-time donation tracking with ETH to INR conversion.
  • Utilization recording with proof uploads and admin approval.
  • Super Admin monitoring of all donations and fund usage.

By combining blockchain integrity with role-based access control (Donor, Trust, and Super Admin), the system eliminates middlemen, reduces corruption, and builds a fully transparent charity ecosystem.

Core Technology & Features

Blockchain-Powered Charity Infrastructure

MetaMask Wallet Integration

Donors send Ethereum donations securely through MetaMask. Each transaction generates a unique blockchain transaction hash that is verified before recording in the system.

Ganache Blockchain Verification

The system connects to the Ganache local blockchain network to fetch and validate transaction details, ensuring the receiver address matches the registered trust wallet.

Django Secure Backend

Built using Django framework with role-based authentication (Donor, Trust, Super Admin) to manage donations, utilization tracking, and system governance.

Web3 Transaction Validation

Web3 integration retrieves blockchain transaction data, validates sender, receiver, and ETH amount, and prevents duplicate or fraudulent entries.

Fund Utilization with Proof

Trusts must record fund usage with purpose, category, and proof upload. Super Admin verifies each utilization before it is approved.

Full Transparency & Monitoring

Donors can track their donations, view transaction history, monitor utilization status, and ensure end-to-end transparency within the charity ecosystem.

  • Donor Registration

  • Blockchain Donation

  • Trust Utilization

  • Admin Monitoring

Donor Registration & Secure Login

Donors create secure accounts to participate in transparent blockchain-based donations.

  • User authentication using Django secure backend.
  • Role-based access (Donor / Trust / Super Admin).
  • Dashboard displaying donation history and transaction tracking.

Once logged in, donors can select verified trusts and proceed to donate securely using MetaMask wallet integration.

Technology Stack
